open fun getItemViewType(element: E): RecyclerSpinnerAdapter.ViewType
Content copied to clipboard

Return the view type of the view used to display the item in the dropdown, for the purposes of view recycling.

The default implementation returns ViewType.Default, fitting a case where a single type of view is used. If multiple kinds of view should be used, override this function to return the appropriate ViewType for each Item
