RecyclerSpinnerAdapter

abstract class RecyclerSpinnerAdapter<I : Any, E, DVH : RecyclerView.ViewHolder, SVH : RecyclerView.ViewHolder>(initiallySelected: I?, differ: DiffUtil.ItemCallback<E>) : ListAdapter<E, DVH>
Content copied to clipboard

A Base class for classes that provides items to be displayed in co.luoja.recyclerspinner.RecyclerSpinners. For most use cases, it is not necessary to inherit from this class, as FlatSpinnerAdapter or NestedSpinnerAdapter are usually sufficient.

See also

androidx.recyclerview.widget.ListAdapter

Parameters

I

: The Type of the items that this RecyclerSpinnerAdapter is used to select.

E

: The Type of the items that are used to define the elements displayed in the spinner dropdown's RecyclerView.

DVH

: The Type of ViewHolder used to manage the views used to display the items in the dropdown.

SVH

: The Type of ViewHolder used to display the selected item in the spinner itself.

initiallySelected

The initially selected item.
