NestedSpinnerAdapter

abstract class NestedSpinnerAdapter<SI : Any, I : Any, SDVH : RecyclerView.ViewHolder, IDVH : RecyclerView.ViewHolder, SVH : RecyclerView.ViewHolder>(initiallySelected: I? = null, differ: NestedSpinnerAdapter.DifferCallback<SI, I>) : RecyclerSpinnerAdapter<I, NestedSpinnerAdapter.Element<SI, I>, RecyclerView.ViewHolder, SVH>
Content copied to clipboard

A RecyclerSpinnerAdapter that presents a two-level hierarchical list of items, where the second level elements can be hidden or displayed.

See also

androidx.recyclerview.widget.ListAdapter

Parameters

SI

: The Type of the objects used to identify sections. (Section Identifier)

I

: The Type of the items that this RecyclerSpinnerAdapter is used to select.

SDVH

: The Type of the ViewHolder used to display headers for each section

IDVH

: The Type of ViewHolder used to manage the views used to display the items in the dropdown.

SVH

: The Type of ViewHolder used to display the selected item in the spinner itself.
